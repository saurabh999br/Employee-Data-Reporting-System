# Employee Data Reporting System

## 📌 Project Description
The **Employee Data Reporting System** is a web-based application developed using **ASP.NET MVC** that retrieves employee information from a **PostgreSQL database** and displays it in a structured table format on the user interface.

The system also provides features to generate **individual PDF reports for employees** and export employee data into an **Excel file**. This helps manage employee records efficiently and allows users to download reports for documentation and analysis.

## 🚀 Features
- Fetch employee data from PostgreSQL database
- Display employee records in a tabular format on the UI
- Generate **PDF reports for individual employees**
- Export employee data into **Excel files**
- Organized **MVC architecture**
- User-friendly interface for data viewing and reporting

## 🛠️ Technology Stack
- **Backend:** ASP.NET MVC (.NET)
- **Database:** PostgreSQL
- **Data Access:** ADO.NET / Npgsql
- **PDF Generation:** iTextSharp
- **Excel Export:** EPPlus
- **Frontend:** HTML, CSS, Bootstrap
- **Development Tool:** Visual Studio

## 📊 Functional Overview
1. The application connects to the PostgreSQL database.
2. Employee data is fetched from the backend.
3. The data is displayed on the UI in a table format.
4. Users can generate **PDF reports for each employee**.
5. Users can export the complete employee dataset into an **Excel file**.
